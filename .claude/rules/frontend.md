# Frontend Rules (Next.js App Router)

- Use Next.js App Router in `src/app`.
- Prefer Server Components by default; use `"use client"` only when needed.
- Use `next/image` for all non-trivial images.
- Use `next/font` for Google fonts; avoid runtime font loaders.
- Keep components small and composable; avoid giant page files.
- No heavy animation libraries by default; prefer CSS and minimal JS.
