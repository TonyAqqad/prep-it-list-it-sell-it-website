# SEO Rules

- Use semantic HTML and correct heading hierarchy.
- Add JSON-LD:
  - `LocalBusiness` (or best-fit schema type)
  - `Service` per service page
  - `FAQPage` for service FAQs
  - `BreadcrumbList` for hierarchical pages
- Use a single canonical URL per page.
- Avoid thin/duplicate pages; only create location pages with distinct value.
- Keep sitemap clean; only include indexable pages.
